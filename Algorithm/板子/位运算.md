// 位运算
//&按位与:按每位,两者全为1才可为1,可以用a&1==1判a的奇偶性
//|按位或:只要一者为1就为1
//^按位异或:相同为0,否则为1(有自反性,x=x^y^y)
//~按位取反:可用~a+1计算相反数
//<<左移:按位左移,右侧补0,左侧溢出位舍去.每左移一位相当于*2
//>>右移:同上,最右位补0(无符号数逻辑位移)或0和1(有符号数算数位移,为负则补1)
// 位运算可从二进制层面调整变量:构造变量充当模板,清空目标变量的特定位,再赋值
// 实际运用：&：判奇偶（取出末位）   特定位清零   x&(x-1)=0
//          |：特定位赋1     模板叠加
//          ^：特定位翻转（0不变1反转）   a^n^n=a
//          ~：取反（含符号位）
//          <<:乘2^n     创建模板
//          >>：循环遍历每一位
//          取特定（几）位：a&=(0B11001100)保留特定位，再右移取出
//          按字节整体处理时，用 0XFF作为基本单元更简单快捷 0XFF即最低八位均为1的模板
//          左右移的运算优先级非常低，要记得用括号括起来
//          &,|运算优先级比等号（不等号）低
